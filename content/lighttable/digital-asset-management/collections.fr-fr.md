Une collection est un ensemble d'images défini par une combinaison spécifique de critères de sélection. Le type de collection le plus élémentaire est la pellicule, qui contient toutes les images qui ont été importées d'un répertoire spécifique du disque.

Vous pouvez facilement construire d’autres types de collections basées sur différents attributs d’image comme les données EXIF, le nom de fichier, les mots-clés, etc. La combinaison logique de critères multiples permet de restreindre ou d’étendre votre collection (voir [filtres de collection](../../module-reference/utility-modules/shared/collect-images.md)).

Pour un accès rapide, darktable garde une liste des collections les plus récemment utilisées (voir [collections récentes](../../module-reference/utility-modules/shared/recent-collections.md)).
