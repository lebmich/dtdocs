# panneau gauche

De haut en bas :

[importer](../module-reference/utility-modules/lighttable/import.md)
: Importer des images depuis le système de fichiers ou depuis un boîtier connecté.

[filtres de collection](../module-reference/utility-modules/shared/collect-images.md)
: Filtrer les images affichées dans le panneau central de la table lumineuse -- aussi utilisé pour contrôler les images affichées dans les modules [pellicule](../module-reference/utility-modules/shared/filmstrip.md) et [chronologie](../../module-reference/utility-modules/lighttable/timeline.md).

[collections récentes](../module-reference/utility-modules/shared/recent-collections.md)
: Montrer les collections d'images récemment utilisées

[informations de l'image](../module-reference/utility-modules/shared/image-information.md) 
: Afficher les informations de l'image

[installer les scripts lua](../module-reference/utility-modules/lighttable/lua-scripts-installer.md) (optionnel)
: Installer les scripts lua

# panneau droit

De haut en bas :

[sélection](../module-reference/utility-modules/lighttable/select.md)
: Sélectionner des images de la table lumineuse en utilisant des critères simples.

[images sélectionnées](../module-reference/utility-modules/lighttable/selected-image.md)
: Effectuer des actions sur les images sélectionnées.

[développement](../module-reference/utility-modules/lighttable/history-stack.md)
: Manipuler le développement des images sélectionnées.

[styles](../module-reference/utility-modules/lighttable/styles.md)
: Stocker l'historique d'une image en tant que style nommé et l'appliquer à d'autres images.

[éditeur de métadonnées](../module-reference/utility-modules/shared/metadata-editor.md)
: Éditer les métadonnées des images sélectionnées.

[mots-clés](../module-reference/utility-modules/shared/tagging.md)
: Étiqueter les images sélectionnées.

[géolocalisation](../module-reference/utility-modules/shared/geotagging.md)
: Importer et appliquer les données de suivi GPX aux images sélectionnées.

[exporter](../module-reference/utility-modules/shared/export.md)
: Exportez les images sélectionnées vers des fichiers locaux ou des services externes.
