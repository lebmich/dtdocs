L'outil comporte trois onglets dans la partie supérieure et un cadre de sortie de texte dans la partie inférieure.

![utilisation](./usage/darktable-chart-usage.png#w75)

Le premier onglet est utilisé pour définir l'image source, le deuxième onglet définit les valeurs de référence (cible) et le troisième onglet contient les contrôles pour générer le style darktable résultant.
