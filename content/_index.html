darktable est une application open source qui offre un flux de travail pour le traitement des photographies. C'est un développeur de fichier raw — une table lumineuse et une chambre noire virtuelles. Il gère vos négatifs numériques dans une base de données, vous les laisse voir au travers d'une table lumineuse zoomable et vous permet de développer et d'améliorer vos images raw.

Le dépôt source de cette documentation peut être trouvé à https://github.com/darktable-org/dtdocs.git. Tout commentaire relatif à cette documentation peut être fourni en créant un ticket ou une pull request sur ce dépôt.

Vous pouvez obtenir ce manuel au format PDF ici.
